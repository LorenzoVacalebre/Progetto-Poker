Generali
avvio la partita e ricevo il le carte per primo -> client invia "carte" per la mano e poi si mette in ascolto della ricezione ovvero stringa con numero;seme per due volte
poi ricevo il flop -> client manda "flop" e riceve "numero;seme/numero;seme/numero;seme"
e infine ricevo il turno -> client invia "turno" e riceve "true" o "false" (stringa tanto converte client)

Avvio
avvio la partita comunico il client -> client invia "client"
quando comunico il client -> client invia subito "carte" e server (appena ha le carte di tutti i giocatori) invia le carte della mano nella maniera specificata prima

quando ricevo le carte allora mi serve il flop -> invio "flop" al server e il server invia a tutti i client le carte del flop nella maniera specificata prima e client aspetta in ascolto  

una volta inviate il flop server subito in ascolto per il tasto "avanti" -> client riceve le due carte e le visualizza una volta visualizzate invia "turno" e server invia il turno solo quando riceve tutti i turno dei client

a questo punto client con il turno ha tre possibilita:
passa: invia "passa" al server
alza puntata : lato client alza la puntata
scommetti: invia "scommetti/puntataScommessa"

client senza turno:
in ascolto fin quando non riceve true 

server in ascolto :
riceve il comando ed esegue l'istruzione citata nel comando
cambia il turno e lo reinvia solo quando client con il turno invia "avanti"
